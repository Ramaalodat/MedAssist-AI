# 🏥 MedAssist AI - نظام التشخيص الطبي الذكي

## 🚀 دليل البدء السريع

### المتطلبات
- Python 3.8 أو أحدث
- مفتاح Google Gemini API ([احصل عليه من هنا](https://makersuite.google.com/app/apikey))

### خطوات التشغيل

#### 1. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

#### 2. تعيين مفتاح API
```cmd
set GEMINI_API_KEY=مفتاح_API_الخاص_بك
```

#### 3. تشغيل الخادم
```bash
python app.py
```

أو استخدم الملف الدفعي:
```cmd
start_server.bat
```

#### 4. فتح الواجهة
افتح `dashboard.html` في المتصفح

## 📋 كيفية الاستخدام

### تحليل الأعراض (New Diagnosis)

1. **افتح صفحة Dashboard** (`dashboard.html`)
2. **اضغط على "New Diagnosis"** من القائمة الجانبية
3. **أدخل الأعراض** في مربع النص، مثال:
   ```
   أعاني من صداع مستمر منذ 3 أيام، حمى 38.5 درجة،
   تعب شديد وآلام في الجسم، وصعوبة في النوم
   ```
4. **اضغط على زر "Analyze"**
5. **انتظر النتائج** (قد يستغرق بضع ثوان)

### النتائج المعروضة:
- ✅ **اسم المرض** - التشخيص المحتمل
- ✅ **درجة الخطورة** - خفيف، متوسط، شديد، حرج
- ✅ **الأسباب المحتملة** - قائمة بالأسباب
- ✅ **العلاجات الموصى بها** - قائمة بالعلاجات

## 🔧 حل المشاكل

### المشكلة: "لا توجد نتائج في قسم Diagnosis Result"

**الحل:**
1. تأكد من تشغيل الخادم:
   ```bash
   python app.py
   ```
2. تحقق من أن الخادم يعمل على المنفذ 8000
3. تأكد من تعيين `GEMINI_API_KEY`
4. افتح console المتصفح (F12) للتحقق من الأخطاء

### المشكلة: "Failed to analyze symptoms"

**الحل:**
- تحقق من اتصال الإنترنت
- تأكد من صحة مفتاح Gemini API
- تحقق من أن الخادم يعمل بشكل صحيح

### المشكلة: "Module not found"

**الحل:**
```bash
pip install -r requirements.txt
```

## 💡 نصائح للحصول على أفضل النتائج

1. **كن محددًا** - اذكر الأعراض بالتفصيل
2. **اذكر المدة** - منذ متى بدأت الأعراض؟
3. **اذكر الشدة** - خفيف، متوسط، شديد
4. **اذكر جميع الأعراض** - لا تترك شيئًا

### مثال جيد:
```
صداع شديد في الجانب الأيسر من الرأس منذ يومين،
يزداد مع الحركة، مصحوب بغثيان خفيف ودوخة،
درجة الألم 7 من 10
```

### مثال سيء:
```

```
