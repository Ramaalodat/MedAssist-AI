# ✅ الخادم يعمل الآن! 🎉

## 🚀 الخطوات التالية

### 1. افتح الواجهة
افتح ملف `dashboard.html` في المتصفح:
- **الطريقة 1:** اضغط مرتين على الملف
- **الطريقة 2:** اسحبه إلى المتصفح

### 2. استخدم النظام

#### أ. تحليل الأعراض (New Diagnosis)
1. اضغط على **"New Diagnosis"** من القائمة الجانبية
2. أدخل الأعراض في مربع النص، مثال:
   ```
   أعاني من صداع شديد منذ 3 أيام
   حمى 38.5 درجة
   تعب وآلام في الجسم
   صعوبة في النوم
   ```
3. اضغط **"Analyze"**
4. انتظر 3-5 ثواني
5. شاهد النتائج! ✅

### 3. النتائج المعروضة
- ✅ **اسم المرض** - التشخيص المحتمل
- ✅ **درجة الخطورة** - خفيف/متوسط/شديد/حرج
- ✅ **الأسباب المحتملة** - قائمة بالأسباب
- ✅ **العلاجات الموصى بها** - قائمة بالعلاجات
- ✅ **تنبيه طبي** - تذكير مهم

## 🛠️ إذا واجهت مشكلة

### المشكلة: "Failed to fetch"
**الحل:** تأكد من أن الخادم يعمل (يجب أن ترى رسالة في Terminal)

### المشكلة: "Analysis Failed"
**الحل:** 
1. تحقق من اتصال الإنترنت
2. تأكد من صحة مفتاح API
3. افتح Console المتصفح (F12) للتفاصيل

### المشكلة: الخادم توقف
**الحل:** شغّله مرة أخرى:
```bash
python app.py
```
